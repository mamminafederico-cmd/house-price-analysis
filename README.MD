# House Price Analysis & Prediction

## Objective
Analyze housing data and build machine learning models to predict house prices.

## Steps
- Exploratory Data Analysis (EDA)
- Feature engineering
- Linear Regression model
- Random Forest model
- Model evaluation (MAE, RMSE, R²)
- Predictions on unseen data

## Key Insights
- Overall quality and living area are the strongest predictors of price
- Random Forest outperforms Linear Regression
- Housing prices show strong non-linear patterns

## Tools
- Python
- pandas
- scikit-learn
- matplotlib
- seaborn

## Models
- Linear Regression (baseline)
- Random Forest Regressor (improved performance)
